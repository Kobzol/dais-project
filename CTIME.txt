1) Data už jsou v databázi, nemusí se nahazovat tabulky ani funkce (případné pořadí je create > functions > init).
2) Pro spuštění unit testů stačí otevřít solution a použít zkratku Ctrl + R, Ctrl + A, anebo v menu zvolit Test > Run > All tests.
3) Výsledky testů lze zobrazit v okně Test explorer (Test > Windows > Test explorer).
